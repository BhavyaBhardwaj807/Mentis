Mentis – Gamified Financial Learning Platform

Mentis is an AI-powered platform designed to make financial education interactive, visual, and engaging. Instead of learning finance through long theoretical explanations, users build financial strategies on a visual canvas and receive intelligent feedback from an AI mentor.

The platform combines visual strategy building, AI reasoning, and gamified challenges to help users understand how financial decisions connect with each other.

Mentis transforms financial literacy from passive reading into active strategy creation and experimentation.

Problem

Financial education today faces several challenges:

• Learning resources are mostly theoretical and text-heavy
• Users struggle to visualize financial strategies
• Chat-based AI tools generate long answers but lack structured guidance
• Beginners often make financial mistakes without understanding their decision-making process

As financial systems become more complex, people need tools that help them practice financial planning in an intuitive and structured way.

Solution

Mentis introduces an interactive approach to financial learning.

Users participate in financial challenges and construct strategies visually by connecting financial concepts such as:

• Budgeting
• Savings
• Investments
• Emergency funds
• Debt repayment

The system analyzes these strategies and provides feedback based on factors like diversification, risk balance, and financial planning structure.

By combining visual thinking with AI assistance, Mentis enables users to learn finance through real decision-making scenarios.

Key Features
Visual Strategy Canvas

Users design financial plans using connected nodes representing financial concepts.

AI Thinking Partner

An AI mentor helps organize ideas, suggest improvements, and guide users through financial planning.

Strategy Evaluation Engine

The platform analyzes strategies based on multiple financial indicators including:

• Diversification
• Risk balance
• Emergency fund presence
• Timeline structure
• Strategy complexity

Gamified Learning Challenges

Users progress through topic-based challenges covering:

• Budgeting
• Saving
• Investing
• Debt management

Strategy Feedback

Users receive scores, strengths, and improvement suggestions to refine their financial strategies.
